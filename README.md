- [TMDB Rest API](https://www.themoviedb.org/documentation/api?language=en-US)

## Installation

React Setup 

    npx create-react-app .

Fetch Date

    npm i axios
    
Runs the app on localhost:3000
    
    npm start
    
Builds the app for production to the build folder.

     npm run build
